# Private Lesson Mentorship — Dallas Arsenal Volleyball Club

A single-page landing site for Dallas Arsenal's private volleyball **mentorship** memberships.

## What's inside
- Three 10-month membership tiers: **The Foundation** (once a month), **Build Momentum** (bi-weekly), and **The Full Program** (weekly).
- A **Pay Monthly ⇄ Pay in Full** toggle — paying the season up front gets one month free (10% off).
- A feature **comparison chart** spanning mentorship & training, performance & nutrition, and premium ("The Full Program") perks like film review and full mentor access.

## View it
Open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Built with
Plain HTML, CSS, and a little JavaScript in a single self-contained file (`index.html`).
Colors and typography are pulled from [dallasarsenal.com](https://dallasarsenal.com): navy `#000068`, gold `#ffa800`, with Montserrat headings and Roboto body text.
